ODPOWIEDZI

1. Określanie właściwości obiektu – dwa sposoby

Sposób 1: notacja kropkowa let osoba = {}; osoba.imie = "Jan"; Sposób 2: notacja nawiasowa let osoba = {}; osoba["imie"] = "Jan"; Notacja nawiasowa jest szczególnie przydatna, gdy nazwa właściwości jest w zmiennej.

2. Odwoływanie się do pól (właściwości) i metod

Odwołanie do pola: osoba.imie; Odwołanie do metody: osoba.przedstawSie(); Zawsze używamy nazwy obiektu, po której następuje kropka i nazwa pola lub metody.

3. Cechy charakterystyczne konstruktora

jest wywoływany automatycznie przy tworzeniu obiektu, posiada taką samą nazwę jak klasa lub funkcja konstruktora, służy do inicjalizacji pól obiektu, używa słowa kluczowego this, nie zwraca wartości (jawnie).

4. Przykład obiektu: trzy pola i jedna metoda

let samochod = { marka: "Toyota", model: "Corolla", rok: 2020, info: function() { return this.marka + " " + this.model; } }; Pola: marka, model, rok Metoda: info()

5. Znaczenie słowa this. Oddzielanie pól i metod

Słowo this wskazuje na aktualny obiekt, na którym została wywołana metoda. this.marka; this.info(); Pola i metody oddzielamy kropką (.), używając składni: this.nazwaPola this.nazwaMetody()

6. Konstruktor w JavaScript – przykład i opis

function Osoba(imie, wiek) { this.imie = imie; this.wiek = wiek; } Opis: Osoba jest funkcją konstruktora, this.imie i this.wiek to pola obiektu, konstruktor służy do tworzenia nowych obiektów na podstawie jednego wzorca.

7. Znaczenie słowa kluczowego new

Słowo kluczowe new: tworzy nowy obiekt, przypisuje this do nowego obiektu, wywołuje konstruktor, zwraca nowo utworzony obiekt. Przykład: let osoba1 = new Osoba("Anna", 25);

8. Znaczenie słowa kluczowego prototype

prototype służy do dodawania właściwości i metod wspólnych dla wszystkich obiektów danej klasy. Osoba.prototype.przywitajSie = function() { return "Cześć, mam na imię " + this.imie; }; BARDZO WAŻNE: Metody umieszczone w prototype NIE są kopiowane do każdego obiektu osobno, lecz są współdzielone, co oszczędza pamięć i poprawia wydajność programu.
