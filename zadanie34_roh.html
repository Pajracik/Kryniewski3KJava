5) Operatory porównania.


1.== (jest równe) 2==3 wynik -> fałsz

2.!= (nie jest równe) 2!=3 wynik -> prawda

3.> (jest większe) 25>3 wynik -> prawda

4.< (jest mniejsze) 2<3 wynik -> prawda

5.>= (większe lub równe) 25>=3 wynik -> prawda

6.<= (mniejsze lub równe) 2<=3 wynik -> prawda

6) Uwaga do operatorów porównania.


Należy odróżnić == ( dwa razy ) od = (jeden raz)

== ( dwa razy ) -> instrukcja PORÓWNANIA

= (jeden raz) -> instrukcja PRZYPISANIA np. a=5

7) Operatory logiczne.

a)&&(i) np.x=3 y=4 (x < 9 && y > 2) wynik -> prawda

b)||(lub) np. x=3 y=4 (x==8 || y==6) wynik -> fałsz

c)!(zaprzeczenie) np.x=3 y=4 !(x==y) wynik -> prawda
